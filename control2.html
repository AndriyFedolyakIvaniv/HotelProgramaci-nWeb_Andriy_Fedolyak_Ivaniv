Reserva

El módulo de reservas permite a los usuarios gestionar y realizar reservas de forma sencilla dentro del sistema. Este proceso es fundamental para garantizar la disponibilidad y correcta asignación de servicios. A continuación, se explican detalladamente los pasos necesarios para realizar una reserva con éxito, así como las opciones disponibles y los posibles errores que pueden surgir.

Paso 1: Acceso al Formulario de Reserva

El usuario debe acceder a la sección de reservas, donde se encontrará con un formulario que permitirá la configuración de los detalles de la reserva. En este punto, es importante proporcionar la información correctamente, ya que esto afectará la disponibilidad y el correcto procesamiento de la solicitud.

Paso 2: Selección de la Persona

El siguiente paso es elegir a la persona para la cual se va a realizar la reserva. Para ello, el sistema desplegará un listado con los nombres de los usuarios registrados, permitiendo la selección del usuario correspondiente. Este paso es esencial para que el sistema asocie la reserva con la persona adecuada y evite confusiones en la gestión.

Paso 3: Configuración de Fechas

El usuario deberá seleccionar la fecha de inicio y finalización de la reserva a través del calendario interactivo proporcionado por el sistema. Es importante asegurarse de que las fechas sean correctas y coherentes con la disponibilidad del servicio o la habitación.

Errores Comunes en la Selección de Fechas

Si el usuario selecciona una fecha de finalización anterior a la fecha de inicio, el sistema generará un mensaje de error, indicando que la configuración es incorrecta. Para corregirlo, el usuario deberá seleccionar una fecha de finalización válida.

No se puede reservar en las mismas fechas de una reserva ya existente

Si el usuario intenta realizar una nueva reserva en las mismas fechas que ya están ocupadas por una reserva existente, el sistema generará un mensaje de error indicando que las fechas seleccionadas ya están reservadas. Para proceder, el usuario deberá seleccionar un rango de fechas diferente y disponible.

Paso 4: Selección del Tipo de Reserva

El usuario podrá seleccionar el tipo de reserva que mejor se ajuste a sus necesidades. Existen diferentes opciones, entre ellas:

  • AD (Alojamiento y Desayuno): Incluye estancia y desayuno.
  • MP (Media Pensión): Incluye estancia, desayuno y una comida.

Paso 5: Confirmación de la Reserva

Una vez que todos los parámetros han sido configurados correctamente, el usuario deberá hacer clic en el botón "Crear reserva". Al hacerlo, el sistema validará la información y, si todo es correcto, procederá a registrar la reserva en la base de datos.

Paso 6: Generación Automática de Factura

Una vez creada la reserva, el sistema generará automáticamente una factura detallada con el costo total de la estancia. Esta factura incluirá un desglose de precios basado en:

  • Duración de la estancia: Número de noches reservadas.
  • Tipo de reserva: Servicios adicionales incluidos.

El usuario podrá revisar la factura antes de proceder con el pago.

Paso 8: Confirmación Final del Precio

Después de generar la reserva y la factura, el sistema mostrará un mensaje con el precio total a pagar. El usuario deberá revisar este monto y confirmar la información antes de proceder con el pago.

Errores Comunes y Soluciones

Durante el proceso de reserva, pueden presentarse algunos errores comunes. A continuación, se detallan los más frecuentes y cómo resolverlos:

  • Fechas incorrectas: Asegúrese de que la fecha de finalización sea posterior a la de inicio.
  • Usuario no encontrado: Si la persona no aparece en la lista, verifique que haya sido registrada correctamente.
  • Tipo de habitación no disponible: Si la opción deseada no está disponible, pruebe con otra fecha o contacte con el soporte.

Conclusión

Siguiendo estos pasos, el usuario podrá completar el proceso de reserva de manera eficiente y sin inconvenientes. En caso de dudas o problemas, es recomendable contactar con el equipo de soporte para recibir asistencia.
